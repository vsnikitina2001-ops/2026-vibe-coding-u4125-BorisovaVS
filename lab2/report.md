# Подключение бота к данным
## Выбор типа интеграции
В этом проекте я подключила Telegram-бота к Excel-файлу с важными датами семьи (`family_dates.xlsx`), чтобы автоматически переносить события в раздел `События` и использовать их для напоминаний.

## Техническая реализация
- Основной стек: `python-telegram-bot`.
- Основное хранилище бота: `data.json` (общие списки и события).
- Источник дат: Excel-файл `.xlsx`, путь задаётся в `.env` через `EVENTS_XLSX_FILE`.
- Импорт выполняется:
  - автоматически при старте бота;
  - вручную командой `/import_events`.
- Защита от дублей: повторный импорт не добавляет одинаковые события повторно.

## Формат Excel
Поддерживаются колонки:
- `Дата` (например, `2026-07-07`),
- `Число`,
- `Месяц`,
- `Событие`,
- опционально поле для количества дней напоминания (если нет — используется значение `1`).

На основе таблицы событие приводится к формату бота:
- `name` — название события,
- `date` — дата в формате `ДД.ММ`,
- `remind_days_before` — за сколько дней напоминать.

- События из Excel успешно загружаются в бот.
