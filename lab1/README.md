# Telegram Family Assistant Bot
Бот-помощник для семьи: помогает вести общие дела, список покупок, уборку по зонам и важные даты.  
Доступ к боту ограничен по whitelist (`ALLOWED_USER_IDS`), данные общие для разрешённых пользователей.

## Возможности
| Команда | Описание |
|--------|----------|
| `/start` | Приветствие и главное меню |
| `/tasks` | Просмотр списка дел |
| `/add_task` | Добавление задач (можно несколько строк) |
| `/shopping` | Список покупок по категориям |
| `/add_item` | Добавление покупок |
| `/cleaning` | План уборки по зонам с прогрессом |
| `/add_cleaning` | Добавление уборки по зоне |
| `/events` | События по сезонам и месяцам |
| `/add_event` | Добавление события |
| `/help` | Краткая справка |
| `/cancel` | Отмена текущего шага |

## Особенности реализации
- Хранение данных в `data.json` (общие списки для всех разрешённых пользователей)
- Доступ только для `ALLOWED_USER_IDS` из `.env`
- Диалоги реализованы через state-машину (`state_set/state_get`), без `ConversationHandler`
- Меню и действия через `ReplyKeyboard` и `InlineKeyboard`
- Напоминания:
  - по задачам и уборке — по заданному времени
  - по событиям — ежедневная проверка с учётом `remind_days_before`
- Автоочистка логов по сроку хранения
- Для действий пользователей поддерживаются метки автора

## Требования
- Python 3.11+  
- Зависимости из `requirements.txt`  
- Токен от [BotFather](https://t.me/BotFather)


## Установка и запуск

Перейдите в каталог с ботом:

```bash
cd "/Users/ххх/Documents/New project"
```

Создайте виртуальное окружение и установите зависимости:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Скопируйте `.env.example` в `.env`:

```bash
cp .env.example .env
```

Укажите в `.env`:

```env
BOT_TOKEN=ваш_токен
ALLOWED_USER_IDS=ваш_id,второй_id,...
DATA_FILE=data.json
TIMEZONE=Europe/Moscow
```

Запустите бота:

```bash
python3 bot.py
```

После запуска в консоли появится сообщение о старте.  
Откройте Telegram, найдите бота и отправьте `/start`.

## Настройка данных
Файл `data.json` содержит:

- `tasks` — список дел
- `shopping` — список покупок
- `cleaning` — план уборки по зонам
- `events` — важные события
- `task_log`, `shopping_log`, `cleaning_log`, `events_log` — журналы действий

Изменения в `data.json` бот подхватывает после следующего запроса/действия.

## Замечания по безопасности
- Не публикуйте файл `.env` и не добавляйте токен в репозиторий
- Токен даёт полный доступ к управлению ботом
- При утечке токена нужно отозвать его через BotFather (`/revoke`) и вставить новый
