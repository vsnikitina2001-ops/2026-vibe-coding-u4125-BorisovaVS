# Telegram Family Assistant Bot

Telegram-бот на Python (`python-telegram-bot`) с функциями:
- умный список дел с приоритетами и напоминаниями;
- список покупок с категориями;
- планировщик уборки по зонам с прогрессом;
- напоминания о важных датах;
- общие списки для всех пользователей из whitelist;
- ограничение доступа по whitelist (`ALLOWED_USER_IDS`).

## Команды
- `/start` — приветствие и меню
- `/help` — список команд
- `/tasks` — просмотр задач
- `/add_task` — добавить задачу
- `/shopping` — список покупок
- `/add_item` — добавить товар
- `/cleaning` — план уборки
- `/add_cleaning` — добавить пункт уборки
- `/events` — список событий
- `/add_event` — добавить событие
- `/cancel` — отмена текущего шага

## Требования
- Python 3.11+

## Запуск
1. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Создать `.env` из шаблона:
   ```bash
   cp .env.example .env
   ```
3. Заполнить `.env`:
   - `BOT_TOKEN` — токен от `@BotFather`.
   - `ALLOWED_USER_IDS` — список user_id, которым разрешён доступ.
4. Запустить бота:
   ```bash
   python bot.py
   ```

## Как узнать свой user_id
- Напиши в Telegram боту `@userinfobot` и возьми поле `Id`.
- Добавь этот `Id` в `ALLOWED_USER_IDS`.

## Хранение данных
Бот хранит данные в JSON-файле (`DATA_FILE`, по умолчанию `data.json`).

Структура включает:
- пользователей;
- задачи (текст, приоритет, статус, время напоминания);
- покупки (название, категория, статус);
- уборку (зона, пункт чек-листа, статус, напоминание);
- события (название, дата, напоминание за N дней).

## Ограничение доступа
Если пользователь не в whitelist, бот отвечает:
`У вас нет доступа к этому боту`.

## Примечания
- Напоминания по задачам и уборке работают в указанное время.
- Для событий запускается ежедневная проверка в 09:00 по `TIMEZONE`.
